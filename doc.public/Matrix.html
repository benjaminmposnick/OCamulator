Module Matrix

module Matrix: sig .. end
type t

t is the type of Matrix. Representation invariants:

  • A Matrix represented as a list of RowVectors
  • All RowVectors which comprise a Matrix are the same length.
val to_list : t -> float list list

to_list mat returns a 2-dimensional list containing the elements of Matrix mat.

val to_array : t -> float array array

to_array mat returns a fresh 2-dimensional array containing the elements of Matrix mat.

val of_list : float list list -> t

of_list lst returns a Matrix containing the elements of the 2-dimensional list lst.

val of_array : float array array -> t

of_array arr returns a Matrix containing the elements of the 2-dimensional array arr.

val make : int -> int -> float -> t

make n m init returns a Matrix that has n rows and m columns, where all entries are initialized to init.

val zeros : ?n:int -> int -> t

zeros (n, m) is a Matrix with n rows and m columns such that each entry is initialized to the float zero. If n is not provided, then the resulting matrix will be m X m.

val identity : int -> t

identity n is a Matrix with n rows and n columns such that the entries along the main diagonal are ones and all other entries are zero. E.g. | 1 0 0 | | 0 1 0 | | 0 0 1 |

val n_rows : t -> int

n_rows mat is the number of rows in mat.

val n_cols : t -> int

n_cols mat is the number of columns in mat.

val is_square : t -> bool

is_square mat is true iff mat has the same number of rows and columns.

val string_of_matrix : t -> string

string_of_matrix mat is the string representation of matrix mat.

val transpose : t -> t

transpose mat is the transpose of mat, obtained by interchanging the rows and columns of mat, i.e. if A =def= mat, then transpose mat = A^T.

val is_symmetric : t -> bool

is_symmetric mat is true iff mat is symmetric, i.e. A = A^T, if A =def= mat. Requires: m is square

val matrix_multiply : t -> t -> t

multiply m1 m2 is the result of performing matrix multiplication between matrices m1 and m2, i.e. if A =def= m1, where A is m X n, and B =def= m2, where B is n X p, then matrix_multiply m1 m2 is AB, where AB is is m X p. Requires: The number of columns of m1 must be equal to the number of row of m2.

val get_row : t -> int -> float list

get_row mat i is the ith row of mat as a list.

val get_col : t -> int -> float list

get_col mat j is the jth column of mat as a list.

val drop_row : t -> int -> t

drop_row mat i is mat without row i.

val drop_col : t -> int -> t

drop_col mat j is mat without column j.

val apply_to_all : (float -> float) -> t -> t

apply_to_all f mat is mat after applying f to all elements in the matrix.

val of_vectors : Vector.t list -> t

of_vectors vec_lst returns a Matrix containing the elements of the vectors in vec_lst. Requires: vec_lst is non-empty.

val is_upper_triangular : t -> bool

is_upper_triangular mat is true iff mat is upper-triangular, i.e. all entries below the main diagonal are zero. Requires: mat is square.

val is_lower_triangular : t -> bool

is_lower_triangular mat is true iff mat is lower-triangular, i.e. all entries above the main diagonal are zero. Requires: mat is square.

val map : (Vector.t -> Vector.t) -> t -> t

map fn mat is mat after applying fn to every row vector in mat.

val map2 : (Vector.t -> Vector.t -> Vector.t) -> t -> t -> t

map2 fn mat1 mat2 is the result from applying fn to all of the corresponding row vectors in mat1 and mat2.

val row_sums : t -> float list

row_sums mat is list of the sums of all of the rows of mat, in the same order they occur in mat.

val matrix_vector_product : t -> Vector.t -> bool -> Vector.t

matrix_vector_product mat vec swap_order is the one of the following: Suppose that A =def= mat and b =def= vec. Then the result is...

  • Ab if swap_order is false and the number of columns of A equals the number of rows of b.
  • bA if swap_order is true and the number of columns of b equals the number of rows of A. If a shape error occurs, then Failure is raised. Requires: The number of columns of m1 must be equal to the number of row of m2.
